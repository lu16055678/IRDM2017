       COMPGI21 - Introduction to Machine Learning

       This database contains the 2017-18 versions of syllabuses. These are still being finalised and changes may occur before the start of the session.

       Syllabuses from the 2016-17 session are available here.

        
         
         
          
         	Code	COMPGI21
	Year	MSc
	Prerequisites	N/A
	Term	1
	Taught By	David Barber (100%)
	Aims	To have a full understanding of the learning outcomes.
	Learning Outcomes	Students will become familiar with the conceptual landscape of machine learning and have developed practical skills to solve real world problems using available software.


 
          
        
 
         
         
         
          
         
          Content

         
 
          
          
         Introduction to Supervised Learning

         	Regression and Classification
	Linear Models for regression and classification
	 Concepts of overfitting and regularization, L1 and L2 regularisation
	Naïve Bayes and Logistic Regression
	Nearest Neighbour classification
	Data preprocessing


          Introduction to Unsupervised Learning

         	Kmeans clustering
	Principal Components Analysis


          Deep Learning

         	Neural Networks for regression and classification
	Autoencoders


          Data Visualisation

         	Using Autoencoders
	tSNE


          Throughout there will be a strong emphasis on practical aspects, with programing exercises.
 
          
        
 
         
         
         
          
         
          Method of Instruction

         
 
          
          
         3 hours of Lectures and 1 hour tutorial per week.
 
          
        
 
         
         
         
          
         
          Assessment

         
 
          
          
         The course has the following assessment components:

         	Written Examination (2.5 hours, 70%)
	Coursework Section (30%)


         To pass this course, students must: 

         	Obtain an overall pass mark of 50% for all sections combined.
	Obtain a pass mark of 50% for the written examination.

 
          
        
 
         
         
         
          
         
          Resources

         
 
          
          
         Flach, P., 2012. Machine Learning The Art and Science of Algorithms that Make Sense of Data. Cambridge: Cambridge University Press
